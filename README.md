# Jogos-em-C
Trabalho da disciplina Algorítimo
O código fonte contém um menu com 3 opções de jogos para serem jogados no terminal, os jogos são: Perguntas e respostas, Cobra na caixa e Gousmas War. Os jogos foram feitos usando comandos como While, If, Else, Switch além de variáveis do tipo Int e Char.
Além disso, foi adicionada a biblioteca time.h para auxílio do jogo 2.
O chatgpt foi utilizado para corrigir erros de código como: local certo do fechamento de chaves nos comandos while e if, nomes de variáveis cujo estavam errados, ausência do & no scanf, e auxílio de como utilizar a biblioteca time.h para sortear aleatoriamente as caixas no jogo 2.
